Este es un BOX MODEL

Es un bloque, se comportará de la siguiente manera:
♦ Tiene content.
♦ Tiene border.
♦ Tiene padding.
♦ Tiene margin.
♦ Cuentan con las propiedades width y height.

Ejemplo de Padding

Padding controla la distancia desde el borde de la caja hasta el contenido de forma interna. Se puede usar de distintas formas: Solo padding da distancia en todas direcciones, padding 12px, 10px aplica distancia VERTICAL y HORIZONTAL, otra opción es por cada lado que se desea dar distancia con padding
